# Audit Standards and Control Layers

This project includes explicit support for international audit and compliance frameworks.

## IFRS as Code

The repository treats IFRS guidance as a programmable accounting control layer:

- Accounting rules are encoded in terms of data processing and validation logic.
- Financial data transformations and synthetic ledger generation are designed to reflect IFRS concepts such as revenue recognition, deferred income, and asset/liability classification.
- Code-based audit checks help make accounting policies reproducible, testable, and reviewable in GitHub.
- Version-controlled artifacts and documentation support transparency and audit traceability.

## EU AI Act as a Compliance Control Layer

The EU AI Act is incorporated as a separate control layer for AI governance and risk management:

- AI scorecards and compliance metrics are calculated to reflect transparency, accountability, and safety requirements.
- Audit workflows include EU AI Act-style governance assessments for model explainability, risk profiling, and anomaly detection.
- The project supports monitoring of AI systems for potential high-risk behavior in financial and cybersecurity contexts.

## International Audit Standards

This repository is aligned with broader audit expectations by emphasizing:

- Cross-border regulatory transparency for IFRS and EU AI requirements.
- Documentation of controls, testable workflows, and issue tracking for audit findings.
- A layered architecture that separates accounting rules, AI controls, and cybersecurity risk assessments.
- Support for auditors through issue templates, workflow artifacts, and n8n automation examples.

## GitHub as an Audit Platform

By hosting IFRS and AI compliance logic in GitHub, this project enables:

- Code reviews for accounting and AI governance logic.
- Audit trails through commit history, pull requests, and GitHub Actions logs.
- Reproducible artifact generation for Excel, CSV, and Word reports.
- Integration with issue tags like `bug4Audit` for structured audit reporting.
